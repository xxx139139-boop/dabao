# 大宝抖音自动中控助手

Chrome 浏览器扩展程序，用于抖音直播间的自动点赞和自动评论功能。

## 📋 功能特性

### ✅ 自动点赞
- 模拟双击直播画面进行点赞
- 可配置每分钟点赞次数（1-60次）
- 随机时间间隔，模拟人类行为
- 智能位置偏移，避免被检测

### ✅ 自动评论
- 支持最多50条预设评论
- 三种发送模式：
  - **随机循环**：随机选择评论发送
  - **按顺序**：按列表顺序逐条循环
  - **智能去重**：避免短时间内重复发送
- 可配置发送间隔（5-3600秒）
- 支持批量导入评论（.txt文件）

### ✅ 友好的 UI 界面
- 固定在页面右侧的侧边栏
- 宽度可调节（320px - 600px）
- 可折叠收起
- 实时操作日志显示
- 使用 Shadow DOM 隔离样式，不影响原页面

### ✅ 数据管理
- 配置自动保存到本地存储
- 统计数据持久化（总点赞数、总评论数）
- 操作日志记录（最多100条）
- 每日自动重置当日统计

## 🔧 安装方法

### 方式一：开发者模式安装（推荐）

1. 下载本项目代码
2. 打开 Chrome 浏览器，地址栏输入：`chrome://extensions/`
3. 开启右上角"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹 `douyin-auto-helper`
6. 扩展程序将自动加载

## 📖 使用方法

1. 打开抖音直播间页面（`https://live.douyin.com/*`）
2. 点击页面右下角的浮动"宝"字按钮打开侧边栏
3. 在侧边栏中配置点赞和评论参数
4. 开启相应功能的开关
5. 点击"保存配置"保存设置
6. 扩展将自动开始工作

### 点赞设置
- 设置每分钟点赞次数范围（建议10-50次）
- 开启开关即可自动点赞

### 评论设置
- 设置发送间隔（建议90秒以上）
- 选择发送模式
- 在文本框中输入评论（每行一条）
- 或点击"导入"按钮从 .txt 文件批量导入
- 开启开关即可自动评论

## ⚙️ 配置参数

| 配置项 | 默认值 | 范围 | 说明 |
|--------|--------|------|------|
| 点赞次数/分钟（最小） | 10 | 1-60 | 每分钟最少点赞次数 |
| 点赞次数/分钟（最大） | 50 | 1-60 | 每分钟最多点赞次数 |
| 评论发送间隔 | 90 | 5-3600 | 两次评论间隔秒数 |
| 评论发送模式 | 随机循环 | - | 随机/顺序/智能 |
| 评论最大数量 | 50 | - | 预设评论条数上限 |
| 侧边栏宽度 | 400 | 320-600 | 侧边栏显示宽度（px）|

## ❓ 常见问题

### Q1: 扩展无法加载？
- 请确保 Chrome 版本 >= 144
- 确保开启了"开发者模式"
- 检查 manifest.json 文件是否存在

### Q2: 点赞/评论没有反应？
- 检查是否正确进入了抖音直播间
- 查看侧边栏中的操作日志，检查错误信息
- 尝试刷新页面后重新加载扩展

### Q3: 如何避免被检测？
- 使用默认的随机时间间隔
- 不要设置过高的点赞/评论频率
- 使用智能去重模式发送评论

### Q4: 数据会丢失吗？
- 配置和统计数据会自动保存到 Chrome 存储
- 即使关闭浏览器也不会丢失
- 卸载扩展后数据将被清除
