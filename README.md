## 安装方法

### 第一步：下载扩展

点击右上角 **Code → Download ZIP**，下载后解压，得到 `douyin-auto-helper` 文件夹。

### 第二步：加载扩展

1. 打开 Chrome 浏览器，地址栏输入：`chrome://extensions/`
2. 右上角开启 **开发者模式**
3. 点击 **"加载已解压的扩展程序"**
4. 选择解压后的 `douyin-auto-helper` 文件夹

### 第三步：开始使用

打开抖音直播间 `https://live.douyin.com`，扩展自动加载，左侧出现控制面板。

## 常见问题

**Q：扩展安全吗？**
A：扩展仅在抖音直播间页面运行，不收集任何个人信息，不影响账号安全。

**Q：会被抖音封号吗？**
A：评论间隔和频率均可自定义，建议设置合理间隔，模拟真实用户行为。
