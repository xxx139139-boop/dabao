大宝抖音全自动中控助手 - 安装使用说明书
================================

【版本】1.0
【适用浏览器】Chrome、Edge、360浏览器等支持Chrome扩展的浏览器

一、插件简介
------------
大宝抖音助手是一款辅助工具，帮助您在观看抖音直播时：
- 自动点赞 - 模拟真人双击点赞行为
- 自动评论 - 按设定时间自动发送评论
- 操作日志 - 记录所有操作记录

二、安装步骤
------------
1. 打开浏览器扩展管理页面
   - Chrome/Edge: 地址栏输入 chrome://extensions
   - 360浏览器: 地址栏输入 se://extensions

2. 开启开发者模式
   - 在扩展页面右上角，找到"开发者模式"开关
   - 将其切换为开启状态

3. 加载已解压的扩展程序
   - 点击左上角"加载已解压的扩展程序"按钮
   - 选择本插件文件夹 douyin-auto-helper

4. 安装完成
   - 插件安装成功后，会在浏览器工具栏显示插件图标
   - 打开抖音直播间，点击图标打开侧边栏即可使用

三、使用方法
------------
1. 打开侧边栏
   - 进入抖音直播间
   - 点击浏览器工具栏的插件图标
   - 右侧会弹出侧边栏

2. 设置自动点赞
   - 找到"自动点赞"开关
   - 设置每分钟点赞次数（默认10-50次）
   - 点击开关启动

3. 设置自动评论
   - 找到"自动评论"开关
   - 设置评论间隔（秒）
   - 在文本框中输入要发送的评论（每行一条）
   - 选择评论模式：随机/顺序/智能
   - 点击"保存配置"

4. 查看日志
   - 在侧边栏底部"操作日志"区域
   - 可查看所有操作记录
   - 直播结束后统计信息也会显示在这里

四、常见问题
------------
Q: 点赞没反应怎么办？
A: 1. 确保已开启自动点赞开关
   2. 确保直播间画面正常显示
   3. 检查浏览器权限是否允许

Q: 评论间隔不生效？
A: 保存配置后需要重新关闭再开启自动评论开关

Q: 如何停止自动操作？
A: 将对应的开关切换为关闭状态即可

五、注意事项
------------
1. 本插件仅供学习研究使用
2. 请合理设置点赞和评论频率
3. 建议单次使用时间不超过2小时
4. 如遇到问题，可查看操作日志排查

六、卸载方法
------------
1. 打开 chrome://extensions
2. 找到"大宝抖音全自动中控助手"
3. 点击"移除"按钮
